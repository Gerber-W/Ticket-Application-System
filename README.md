# Ticket Application System

## Overview  
This is a C# Windows Forms group project. The application allows users to enter passenger details and calculate a ticket price based on distance and travel category. The system applies business rules and displays a final ticket summary.

## Features  
The system captures passenger name, age, gender, category, and travel distance. It calculates ticket prices using category-based rates and applies validation to ensure correct input.

## Pricing Rules  
Category One: R20 Per km  
Category Two: R35 Per km  
Category Three: R50 Per km  

## Discount Rules  
Passengers under 12 receive a free ticket.  
Female passengers receive a 50 percent discount on the final ticket price.
